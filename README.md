## About This Repository

This repository documents my solutions to the **HackerRank 30 Days of Code** 
challenge — a curated series of daily programming problems designed to build a solid foundation in core computer science concepts.
Every solution is written in **Java** with a focus on clarity, efficiency, and clean code practices.

## Key Concepts Covered

```
- Variables & Data Types          - OOP (Inheritance, Abstraction)
- Control Flow & Loops            - Interfaces & Generics
- Functions & Recursion           - Linked Lists & Trees
- Arrays & 2D Arrays              - Queues & Stacks
- Maps & Dictionaries             - Sorting Algorithms
- Binary Numbers & Bitwise Ops    - Time Complexity (Big-O)
- Exception Handling              - Regular Expressions
```
